# Wedding Invite — upute

1) Otvori ovaj folder u Visual Studio Code.
2) Dvoklik na `index.html` da vidiš stranicu ili koristi ekstenziju **Live Server** (desni klik → Open with Live Server).
3) Zamijeni slike u `assets/img/` svojim:
   - `hero.jpg` — glavna crno-bijela fotka
   - `cathedral.jpg` — lokacija ceremonije
   - `couple.jpg` — zadnja fotka
   (Nazive možeš promijeniti, ali onda u `index.html` zamijeni putanje.)
4) U `index.html` promijeni tekst (imena, lokacija, vrijeme). Link za Maps i RSVP zamijeni svojim.
5) U `script.js` promijeni datum u `eventDate`.
6) Objavi besplatno:
   - GitHub Pages (Settings → Pages → Deploy from branch → main/root)
   - Netlify (New site from Git → publish dir: root)
